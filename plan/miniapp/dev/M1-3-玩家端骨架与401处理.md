## 2. 401 行为（与后端对齐）

| 场景 | 行为 |
|------|------|
| HTTP **401** 且请求需鉴权（默认 `auth: true`） | `clearStoredTokens` → toast → **`reLaunch` `/pages/auth/login`** |
| 业务码 **40101**（HTTP 仍为 2xx 的极端情况） | 同上 |
| 已在登录页 | 不再 **reLaunch**（防循环） |
| `POST /auth/wechat/login`（`auth: false`） | **不**走 `handleUnauthorized` |
