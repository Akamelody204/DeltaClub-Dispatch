# P0-4 · 微信审核自检表（v0）

> **对应任务**：产品负责人工作计划 · Phase 0 · P0-4。  
> **用途**：**每次提审前**产品过一遍；**不替代**微信官方规则，以 [微信公众平台](https://mp.weixin.qq.com/) 当前公示、类目审核意见为准。  
> **关联**：`P0-3-一期范围与不做清单.md`（一期不做支付、默认不做客服外链）；`P0-1-一期业务流程说明（demo）.md`（主流程与文案边界）。

---

## 0. 使用说明

| 项 | 说明 |
|----|------|
| **谁执行** | 产品主责；小程序开发配合核对实现与文案是否一致。 |
| **何时执行** | 首次提审前必做；**每次**提交审核前复做（含代码包版本变更、文案/页面增删）。 |
| **结果处理** | 任一条「不通过」须修复或取得负责人书面豁免后再提审；豁免项记入 §8。 |
| **版本** | 本文 **v0**：随官方规则变化由产品升版（v0.1、v1.0…）并记 §9 修订记录。 |

## 3. 内容与合规：导流、交易、联系方式

| 序号 | 检查项 | 通过 |
|------|--------|------|
| 3.1 | 无 **诱导添加个人微信/QQ/群**、无 **二维码** 引流到站外交易（除非合规且已评估） | ☐ |
| 3.2 | 无 **引导私下转账/线下付款完成平台内订单** 的明确话术（一期为 **展示价 + 线下结算**，见 P0-3；页面不得做成「扫码付给我」类） | ☐ |
| 3.3 | 无 **绕过小程序交易** 的链接或按钮（与 P0-3「不做微信支付」一致，亦勿引导去外部店铺下单） | ☐ |
| 3.4 | 无 **虚假、欺诈、过度承诺**（如「必赢」「包上分」等无法兑现表述，按俱乐部合规要求自查） | ☐ |
| 3.5 | 无 **违法违规游戏服务** 宣传（代练封号、外挂等若涉及须一票否决并改文案） | ☐ |
| 3.6 | **客服入口**：与 **P0-3 §4.2** 一致——默认 **无外链客服**；若已书面改为 B，再逐项对照平台对 **客服/客服消息** 的要求 | ☐ |

## 4. 页面与文案抽检（主路径）

按 **P0-5** 关键页至少各扫一遍；下列为必查句式。

| 序号 | 检查项 | 通过 |
|------|--------|------|
| 4.1 | **首页/下单页**：服务描述为「信息服务/陪练预约」等中性表述，避免敏感词堆砌 | ☐ |
| 4.2 | **订单/价格**：展示为「参考价」「预估」或俱乐部定价说明，**不宣称**「平台担保交易」若实际无担保 | ☐ |
| 4.3 | **空状态/错误提示**：无辱骂、无歧视、无违规梗 | ☐ |
| 4.4 | **分享转发**（若开放）：标题/描述无违规、无诱导点击欺诈 | ☐ 不适用 / ☐ |
